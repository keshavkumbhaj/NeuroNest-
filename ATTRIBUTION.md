# AI Model

## Qwen2.5-3B-Instruct

- Developed by Alibaba Cloud's Qwen Team
- Used as the on-device language model for generating personalized coaching.
- Model Format: GGUF
- Quantization: Q4_K_M

---

# AI Runtime

## llama.cpp

- Open-source C/C++ inference engine for running Large Language Models locally.
- Used to perform fully offline inference on CPU.

Repository:
https://github.com/ggml-org/llama.cpp
